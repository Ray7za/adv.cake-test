# Установка

### Если запускаете впервые:

- `git clone https://github.com/Ray7za/adv.cake-test.git`
- `cd adv.cake-test`
- `docker compose up -d --build`
- `docker compose exec php bash`
- `composer install`

### Если запускали ранее:
- `docker compose up -d`


# Заметки
